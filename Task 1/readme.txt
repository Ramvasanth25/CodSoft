# Flutter App - Quote of the Day

A simple and elegant Flutter app that displays a new inspiring quote every day. Users can share, favorite, and view their favorite quotes easily.

## ✨ Features

- 📝 Daily inspirational quotes
- 💾 Save quotes to favorites
- 🔁 Refresh to get a new quote
- 📤 Share quotes with friends
- 🌙 Beautiful, responsive UI with light/dark theme support

## 🚀 Getting Started

These instructions will help you get a copy of the project up and running on your local machine for development and testing.

### Prerequisites

Make sure you have Flutter and Dart installed. You can verify using:

```bash

flutter pub get

Run the app
