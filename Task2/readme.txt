# ✅ Flutter To-Do List App

A simple and intuitive To-Do List mobile app built using Flutter. This app allows users to add, update, delete, and manage their daily tasks effectively.

## 📱 Features

- ➕ Add new tasks
- 📝 Edit existing tasks
- ✅ Mark tasks as complete or incomplete
- 🗑️ Delete tasks
- 📅 Add due dates and priorities (optional)
- 🌙 Clean, user-friendly UI with light/dark mode support

## 🛠️ Built With

- [Flutter](https://flutter.dev/)
- [Dart](https://dart.dev/)
- [Provider](https://pub.dev/packages/provider) – for state management *(optional)*
- [Shared Preferences](https://pub.dev/packages/shared_preferences) – to persist tasks *(optional)*

---

## 🚀 Getting Started

Follow these steps to run the app on your local machine:

### 📦 Prerequisites

- Flutter SDK installed  
- Android Studio / VS Code setup  
- Emulator or physical device connected

# To Get dependencies

flutter pub get


# To Run the app

flutter run
